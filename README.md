## 一、要求

### 名词声明

1. **命题变元** 
    - 字母及数字构成的字符串且首字符是字母 
    - 字符串长度不超过 **10** 

1. **第一类逻辑联结词**: 
    - 0,1,¬,∧,∨,→,⊕,↔ 

1. **第二类逻辑联结词(新定义逻辑联结词)**: 
    - 字母及数字构成的字符串且首字符是字母 
    - 给出逻辑联结词的元数及真值表的最后一列 
    - 一共不超过 **10** 个新定义逻辑联结词
    - 新定义逻辑联结词**元数**不超过 **10**
